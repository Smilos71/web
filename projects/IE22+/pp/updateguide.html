How to update IE22++

Keep your software running smoothly

  1. Step 1: Download Get the latest version of the software.
    Click Here to download the updater
  2. Step 2: Run the installer Locate and double-click the downloaded .exe file to start the process.
    3. 3. IMPORTANT: Close all running instances of IE22++ before proceeding!
  3. Step 4: Update In the installer window, simply click the Update button to begin.
  4. Step 5: Wait The system is performing the update. This usually takes less than a minute.
  5. Step 6: Open IE22++ Launch the application again from your desktop or start menu.
  6. 7. Done! You are now using the latest version.
