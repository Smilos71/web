Captify - Commands

EVERYONE

  • nothing :(

ADMIN ONLY

  • /ban add - Add a user to the banned list
  • /ban remove - Remove a user from the banned list
  • /ban temp - Temporarily ban a user for a set time
  • /captcha-text - Sends a message where users have to solve a captcha text
  • /channel lock - Lock a channel to prevent messages from being sent
  • /channel unlock - Unlock a channel to allow messages to be sent again
  • /classic-captcha - A classic captcha where you click on the checkmark emoji to gain a role
  • /kick - Kick a user from the server
  • /purge - Delete a specific number of messages from a channel (same as /clear)
  • /say - Make the bot send a message as if it came from the user
  • /slowmode - Set slow mode in a channel, limiting how often users can send messages
  • /ticketsystem-setup - Set up a ticket system for user support
  • /warn - Warn a user for a specific reason
